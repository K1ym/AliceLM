## 6. Design System & Aesthetic Guidelines

### 6.1 Design Philosophy

**Core Principle**: Clean, rounded, spatial design (Reference: Lovart)

| Principle | Description | Implementation |
|-----------|-------------|----------------|
| **Soft & Rounded** | Generous border-radius creates friendly, modern feel | `rounded-xl` to `rounded-2xl` for containers |
| **Floating Elements** | Key UI floats above content with subtle shadows | Sidebar, modals, dropdowns |
| **White Space** | Abundant padding, content breathes | `p-6` minimum, `gap-6` between cards |
| **Grayscale + Subtle** | Mostly neutral, with very subtle warm accents | `neutral-*` palette, minimal color |

### 6.2 Typography System


| **Body** | `text-sm md:text-base font-light leading-relaxed` | Paragraphs |
| **Caption** | `text-xs text-muted-foreground` | Timestamps, metadata |
